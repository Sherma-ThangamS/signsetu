## ğŸ“¦ API Endpoints

### `GET /words`

- Returns a list of all words.

### `GET /words/:query`

- Searches for a word (case-insensitive, regex match on word or definition).

### `POST /words`

- Adds a new word.

```json
{
  "word": "Learn",
  "definition": "To gain knowledge",
  "imageUrl": "https://...",
  "videoUrl": "https://..."
}
```

### `PUT /words/:id`

- Updates an existing word by ID.

### `DELETE /words/:id`

- Deletes a word by ID.
